# Representative outputs

This folder contains representative outputs from the Aspen Plus runs
to enable verification of the reported results.

## File: DNMC_Output result.xlsx

This Excel file consolidates the key output data generated from the Aspen Plus
process simulations and subsequent techno-economic and environmental analyses.

### Sheet descriptions
- `2. Aspen_stream info`  
  Key Aspen Plus stream tables including mass flow rates, compositions,
  temperatures, and pressures.

- `2-1. Aspen Utility`  
  Utility duties and energy demands calculated by Aspen Plus.

- `2-2. Equipment cost`  
  Equipment-level cost calculations used for techno-economic evaluation.

- `3. UPC & MSP (fixed)`  
  Unit production cost and minimum selling price under fixed assumptions.

- `4-1. UPC & NCE (Grid)`  
  Cost and net carbon emission results assuming grid electricity.

- `4-2. UPC & NCE (RE)`  
  Cost and net carbon emission results assuming renewable electricity.

- `5. Sensitivity analysis`  
  Representative sensitivity analysis results for key process and economic parameters.


## Notes
- This single Excel file is intentionally used to improve accessibility
  and transparency for reviewers and readers.
- These outputs are provided so that reviewers/readers can verify the reported
  results even if they do not have access to Aspen Plus.
